Flujo de control

El control flow (flujo de control) en JavaScript describe cómo se ejecutan las instrucciones de un programa, desde el inicio hasta el final. Se basa en estructuras como secuencias, condiciones y bucles que determinan qué parte del código se ejecuta y cuándo.

  • Usar la declaración if-else.
  • Usar una declaración switch.
