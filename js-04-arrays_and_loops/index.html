JS 04

Arreglos y bucles

Los arrays son un tipo de objeto en JavaScript. Los arrays son una estructura de datos que contiene una lista de elementos.

Los bucles For se utilizan para iterar a través de una secuencia/array de tipos de datos u objetos. Su ejecución depende de una condición booleana (verdadera o falsa) - si la condición es verdadera, entonces el bucle continúa ejecutándose, pero si la condición es falsa, entonces el bucle se rompe.

Una sentencia while crea un bucle que ejecuta una instrucción específica siempre que la condición de prueba se evalúe como verdadera.

Objetivos de aprendizaje:

  • Definir y usar arrays.
  • Acceder a un elemento del array en una posición determinada.
  • Agregar un nuevo elemento a un array.
  • Eliminar un elemento de un array.
  • Usar los diferentes bucles de JavaScript: While loop y For loop
  • Explicar la diferencia entre un bucle For y un bucle While.
