Manipulación del DOM

El DOM (Document Object Model) es una interfaz de programación que representa un documento HTML o XML como una estructura jerárquica de nodos. Actúa como un puente entre el código fuente HTML y los lenguajes de programación como JavaScript, permitiendo acceder, modificar y manipular dinámicamente los elementos de la página web.

Los operadores son símbolos que permiten realizar operaciones sobre uno o más valores (operandos) para producir un resultado.

Los operadores y expresiones en JavaScript permiten manipular datos, realizar cálculos y tomar decisiones en el código. Son esenciales para construir lógica y funcionalidad en cualquier programa.

Objetivos de aprendizaje:
  • Explicar el DOM.
  • Usar la API DOM para acceder a elementos HTML (document.getElementById).
  • Acceder y modificar una clase y atributos de objeto DOM.
  • Leer y validar datos de formulario usando JavaScript.
  • Ocultar y mostrar elementos HTML usando la propiedad de visualización.
