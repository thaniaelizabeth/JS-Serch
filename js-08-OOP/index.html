Programación Orientada a Objetos

La Programación Orientada a Objetos (POO) es un paradigma de programación que se basa en la creación y manipulación de objetos. Un objeto es una unidad que combina características (atributos) y funciones (métodos) que operan sobre esas características.

La POO se basa en cuatro principios fundamentales:

  1. Encapsulamiento: Restringe el acceso directo a algunos componentes.
  2. Abstracción: Oculta los detalles internos y expone solo lo esencial.
  3. Herencia: Permite crear nuevas clases basadas en clases existentes, promoviendo la reutilización del código.
  4. Polimorfismo: Permite que diferentes objetos respondan al mismo método de distintas maneras, facilitando la flexibilidad y extensibilidad del código.
Objetivos de aprendizaje:
  • Los principios SOLID.
  • Definir una clase con atributos y métodos.
  • Instanciar una clase e invocar sus métodos.
  • El formato JSON y su importancia.
  • Definir objetos JavaScript usando el formato JSON.
